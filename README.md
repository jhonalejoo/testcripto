# Crypto Test

Aplicación en React Native (iOS y Android) para listar criptomonedas en USD.

## Instalación

```bash
git clone https://github.com/jhonalejoo/testcripto
cd testcripto
npm install
```

### Android

```bash
npx react-native run-android
```

### iOS

```bash
cd ios
pod install
cd ..
npx react-native run-ios
```
